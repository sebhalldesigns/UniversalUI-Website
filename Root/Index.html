Overview

UniversalUI is a powerful, efficient and flexible app framework.

Using a clean-slate approach, it supports multiple different design paradigms, providing an intuitive and clean API that can be used for any kind of cross-platform app.

Supported Platforms

    Desktop

  • Microsoft Windows - 7 onwards
  • macOS - 10.11 onwards
  • Linux/other - X11 & OpenGL required

    • Mobile

  • iOS & iPadOS - 10 onwards
  • Android - 9 onwards

    • Web

  • Browsers supporting WASM & WebGL


Supported Languages

    All Platforms

  • C / C++
  • Rust

    • Desktop

  • Python
  • C#

    • Desktop & Web

  • JavaScript
  • TypeScript


Supported paradigms

  • Explicit, direct layouts using constraints & manual positioning
  • XML descriptive layouts, with hot reloading
  • Graphical components alongside XML
  • Separate style sheets for theming
